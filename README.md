# Phishing URL Detection System Using FA and Regex

A cybersecurity project that detects phishing URLs using Regular Expressions (Regex) and Deterministic Finite Automata (DFA). The system analyzes URL patterns, suspicious keywords, and domain structures to identify potentially malicious websites.

## Features
- Regex-based URL analysis
- DFA state transition model
- Suspicious keyword detection
- Malicious domain extension detection
- Phishing alert generation

## Technologies
- Python
- Regular Expressions
- Finite Automata
- JFLAP / Graphviz
